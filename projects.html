WorkRoute: Real-Time Employee Tracker & Alert System
A location and emergency alert system for workers to update managers about delays, traffic, or emergencies. Built with HTML, CSS, JavaScript, and Flask for smooth real-time updates. Integrated with Google Maps API for accurate location tracking. Designed with a user-friendly dashboard for both employees and managers.
Movie Revenue Prediction
A machine learning project using Gradient Boosting Regressor to predict movie box office revenue. Features like budget, runtime, and genre are used for better prediction accuracy. Achieved over 85% prediction accuracy using cross-validation. Data visualization tools like Matplotlib and Seaborn were used for insights.
